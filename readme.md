# BoogeeDoo-Launcher

> A Minecraft Launcher, build with electron, React and :heart:

## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
